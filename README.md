# Kelevra Corp - Landing Page

Landing page premium para a **Kelevra Corp**, focada em reabilitação de crédito e soluções jurídicas.

## Tecnologias Utilizadas
- HTML5 Semântico
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Montserrat & Playfair Display)
- Font Awesome (Ícones)

## Estrutura do Projeto
- `index.html`: Estrutura principal do site.
- `style.css`: Estilização e design system.
- `script.js`: Animações e interações.
- `assets/`: Pasta para imagens e recursos visuais.

## Como Visualizar
Basta abrir o arquivo `index.html` em qualquer navegador moderno.
